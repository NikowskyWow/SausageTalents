* **Version**: v1.1.0
* **WoW**: 3.3.5a

## ‚ú® Features

* **Unified Profiles:** Saves your **Talent Tree** allocation and **Action Bar** layout into a single preset.
* **Cross-Realm & Cross-Character:** Seamlessly import profiles from other characters on different realms (e.g., from Onyxia to Blackrock).
* **Smart Talent Learning:** Automatically calculates missing points and learns them instantly.
* **Conflict Protection:** Checks if your current talents conflict with the saved profile.
* **Modern UI:**
    * Dark, themed interface matching the "Sausage" design suite.
    * **Class Validation:** Prevents loading incompatible profiles.
    * **Live Progress Bar:** Visual feedback during profile application.
    * **Safety Features:** Confirmation dialog before deleting profiles.

## ‚ö†Ô∏è Important Note About Talents

The addon **cannot unlearn talents** for you. In WotLK, talent points are permanent until you visit a Class Trainer.

* If you have **conflicting talents** (e.g., you have 5 points in *Divine Strength* but the profile requires 0), the addon will stop and show an error message.
* **Solution:** Visit a Class Trainer, reset your talents, and then apply the profile again.

## üéÆ How to Use

### Basic Usage
1.  Open the menu via **Minimap Icon** or type `/st`.
2.  **Save a Profile:** Type a name in the text box (e.g., "PvP Retri") and click **Save**.
3.  **Load a Profile:** Select a profile from the list (it will highlight blue) and click **APPLY SELECTED**.

### Cross-Realm Import
You can easily copy a build from a leveled character to a new one.

1.  Open Sausage Talents (`/st`).
2.  Use the **Dropdown Menus** at the top left and right.
3.  Select the **Realm** and **Character** you want to copy FROM.
4.  Select the profile from the list and click **APPLY SELECTED**.
